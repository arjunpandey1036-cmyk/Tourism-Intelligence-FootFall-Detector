What This Project Solves

Tourists avoid overcrowded hotspots using live crowd levels, best-time recommendations, nearby low-crowd alternatives, crowd forecast, and smart itinerary planning.

Hackathon Demo Value

  • Scenario simulator: Normal, Weekend Surge, Festival Peak
  • Impact KPIs: avoided overcrowded spots, wait-time saved, diversion success
  • One-click demo story for judges with guided flow

Core Features

  • Live check-ins with real-time crowd levels
  • Crowd forecast for next 3 hours per place
  • Smart itinerary generation with time window optimization
  • Review + rating + photo-backed social trust signals

Tech Stack

  • Frontend: HTML, CSS, JavaScript (Vanilla)
  • Backend: Node.js + Express
  • Database: MongoDB + Mongoose
  • Maps: OpenStreetMap + Leaflet + Heatmap
